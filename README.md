# Sabor do Brasil -  

**Sabor do Brasil** é um projeto educativo desenvolvido como parte do SAEP (Sistema de Avaliação da Educação Profissional), que tem como objetivo proporcionar uma experiência prática aos alunos, integrando conhecimento técnico e criatividade em um ambiente profissional simulado.  

## 🚀 Principais Recursos  

### 🔐 Acesso Seguro  
- **Login exclusivo** 
  **Credenciais para teste:**  
  - 📧 E-mail: `ana@example.com` | 🔑 Senha: `senha123`  
  - 📧 E-mail: `luiza@example.com` | 🔑 Senha: `123456`  

### 👤 Perfil Personalizado  
- Exibe foto, nome e histórico de interações (likes/dislikes recebidos).  
- Modo "Empresa" quando não logado, mostrando dados institucionais.  

### 🍽️ Cardápio Digital  
- Galeria de pratos típicos brasileiros com:  
  - 🖼️ Imagem ilustrativa  
  - 📍 Localidade de origem (cidade/estado)  

### ❤️ Interação Dinâmica  
- **Sistema de likes/dislikes** com contagem em tempo real.  
- **Comentários** (postar, editar e excluir).  

### ⚡ Experiência Fluida  
- Atualizações automáticas sem recarregar a página.  
- Modal de login com validação intuitiva.  

## 💻 Stack Tecnológica  
| Front-end          | Back-end       | Banco de Dados |  
|---------------------|----------------|----------------|  
| HTML5, CSS3, Bootstrap | ASP.NET Core   | MySQL          |  
| JavaScript           | Entity Framework | API RESTful   |  

## 📌 Guia Rápido  
1. **Preparação do Banco**: Execute o script SQL da pasta `/script`.  
2. **Acesso**: Abra o site no navegador.  
3. **Login**: Use uma das credenciais de teste.  
4. **Explore**:  
   - Interaja com pratos (curtir/comentar).  
   - Monitore seu desempenho no perfil.
